### How to run app:
Make sure you're using node > 12 (for expo cli).  
Run:
```
$ nvm use
```
It's being taken from `.nvmrc` file (you can install project version by `nvm install 12.14`).
##### First run:
Install expo globally:
```
$ npm install -g expo-cli
```

##### Run:
```
$ npm install
$ npm start
```
